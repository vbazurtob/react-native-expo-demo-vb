1. Introduction

This application (react-native-expo-demo-vb app and will be referred from here just as “App”) is a demonstration project created to showcase minimal development capabilities using React Native and Expo.

Your privacy is important to us. This App does not collect, store, process, or share any personal data from users.

2. Purpose of the App

The App is intended solely as a technical demo. It demonstrates:

  • Displaying a static list of fictional ID badges
  • Fetching and displaying public RSS news content

It is not intended for commercial use, user accounts, or data processing activities.

3. Information We Collect

No Personal Data Collection

  • No login or registration required
  • No personal information collected
  • No user data stored
  • No access to location, contacts, camera, microphone, or device storage
  • No analytics or tracking technologies used
  • No advertising SDKs integrated

No user or device information is collected, transmitted, or retained.

4. App Functionality

Static ID Badge List

The App displays a static list of randomly generated, fictional ID badges (name, last name, initials, and member ID). All displayed information is fictitious and not linked to real individuals.

Public News Feed

The App retrieves and displays publicly available RSS news content from the official public news feed of The White House.

The App does not modify the content and does not track user interaction with the news feed. No personal data is transmitted when retrieving publicly available RSS content.
